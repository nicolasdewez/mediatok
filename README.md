mediatok
========

## Requirements 

* Docker
* Docker Compose
* Docker Repository: iamluc/host-manager


## Start application

Use this command:

```bash
make start
```

## Access to application

You can go to: [http://web.mediatok/](http://web.mediatok/)

Dev environment is used by default.
